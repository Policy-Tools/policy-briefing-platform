# policy-briefing-platform
Prototype platform for AI-assisted (Dutch) policy briefings based on open-source information. 

Concept prototype for a policy intelligence platform that collects open-source information (news, policy documents, and government sources) and generates short policy briefings and audio updates. 
The goal of this prototype is to explore the structure, interface and information architecture of the platform before implementing AI systems and automated data collection. 

This repository currently contains:
- basic website structure
- concept interface
- example briefings
